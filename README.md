
# 📱 Mobile Python & AI Editor

ブラウザだけで完結する、モバイルファーストなAIアシスト付きPythonエディタです。
サーバーや環境構築は一切不要。HTMLファイルをブラウザで開くだけで、本格的なPythonの実行からGeminiによるAIコード生成、複数ファイルの管理までを行うことができます。

## ✨ 主な機能

### 🐍 完全ブラウザ動作のPython環境
- **Pyodide搭載**: WebAssembly技術により、ブラウザ上で直接Pythonコードを実行します。
- **自動パッケージインストール**: コード内の `import` 文を自動検知し、実行前に必要なパッケージ（`numpy`, `pandas`, `scikit-learn`など）を `micropip` で自動インストールします。
- **環境変数 (.env) サポート**: 実行時に `os.environ` へ自動でロードされる環境変数を設定画面から定義可能です。

### ✨ AIコーディングアシスタント (Gemini API統合)
- **インラインAI指示**: エディタ内に `# AI: 1から10まで足す関数` とコメントを書き「✨ AIに頼む」ボタンを押すと、その場にコードを生成して挿入します。
- **コンテキスト理解**: 現在開いているファイルのコード全体を文脈としてAIに送信するため、精度の高いコード補完が可能です。
- **履歴管理**: 過去にAIに指示したプロンプトと生成結果の履歴をローカルに保存し、いつでも検索・参照できます。

### 📂 プロ品質のファイル & ワークスペース管理
- **ローカルフォルダの直接編集**: File System Access API を活用し、PCのローカルフォルダを直接ブラウザで開いて編集可能です。
- **仮想ファイルシステム**: IndexedDBを利用し、作成したファイルはブラウザ内に自動保存されます（オートセーブ機能搭載）。
- **複数タブ対応**: 複数の `.py` ファイルを作成し、タブでシームレスに切り替えられます。
- **ZIPエクスポート/インポート**: ワークスペース全体（ソースコード、AI設定、環境変数、pip依存関係）を `workspace_backup.zip` として一括保存・復元できます。

### 📱 モバイルフレンドリー UI/UX
- スマホ画面でも操作しやすいレスポンシブなレイアウト。
- **CodeMirror 5**: 構文ハイライト（Darculaテーマ）、行番号表示、自動インデントをサポート。
- スライドイン式サイドバーによる省スペースなファイルエクスプローラー。
- **WakeLock API**: AIの生成待機中などにスマホの画面がスリープするのを防止します。

## 🛠️ 技術スタック

- **UI/Editor**: HTML5, CSS3, JavaScript (Vanilla), [CodeMirror 5](https://codemirror.net/5/)
- **Python Engine**: [Pyodide](https://pyodide.org/) (WebAssembly)
- **AI Integration**: Google Gemini API (REST直接呼び出し)
- **Storage**: IndexedDB (設定・履歴・仮想ファイル保存)
- **File System**: File System Access API, [JSZip](https://stuk.github.io/jszip/)

## 🚀 使い方

### 1. 起動方法
サーバーの構築は不要です。提供されている `index.html`（ファイル名は任意）をダブルクリックして、最新のモダンブラウザ（Chrome, Edge, Safari推奨）で開くだけで起動します。

### 2. 初期設定 (AI機能を利用する場合)
1. 画面右上の「⚙️ 設定アイコン」をクリックします。
2. **AI設定タブ** で利用するモデル（デフォルト: `Gemini-2.5-flash`）を選択します。
3. Google AI Studioで取得した **APIキー** を入力し、「保存」を押します。
   > 🔒 **セキュリティ**: APIキーはサーバーに送信されることなく、お使いのブラウザの IndexedDB にのみ安全に暗号化（隔離）されて保存されます。

### 3. AIにコードを書かせる
- **方法A（上部入力欄）**: 画面上部のテキストボックスに指示（例: `現在のコードのバグを直して`）を入力し、「✨ AIに頼む」を実行します。コードの末尾に結果が追記されます。
- **方法B（インライン指示）**: エディタ内の任意の行に `# AI: 指定のディレクトリ内のファイルをリストアップする関数` と記述し、「✨ AIに頼む」を実行します。コメントの直下にコードが挿入されます。

### 4. 外部パッケージの利用
`import requests` や `import numpy` などをコードに書くだけで、実行時にシステムが自動検出してインストールします。
また、設定の **pip (依存) タブ** に `requirements.txt` のようにパッケージ名を列挙しておくことも可能です。

## ⚠️ 制約・注意事項

- **ローカルフォルダ機能**: 「📂 ローカルフォルダを開く」機能は、File System Access API をサポートしているブラウザ（Chrome, EdgeなどPC環境）でのみ機能します。iOS Safariなど非対応の環境では、仮想ファイルシステム（ブラウザ内保存）として動作します。
- **通信環境**: AIの利用およびPyodide/パッケージの初回ロードにはインターネット接続が必要です。
- **APIモック**: コード内の設定にある GPT-4o, Claude 3 などのモデルは、UI上のプレースホルダー（Mock）です。実稼働させるには対応するAPIエンドポイントとロジックの追加実装が必要です（現在はGeminiのみ完全対応）。
